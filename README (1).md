## Description

## Setup Instructions
1. Navigate to the project folder:
   ```bash
   cd resume_app
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   streamlit run app.py
   ```
